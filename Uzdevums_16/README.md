# 🎯 Uzdevums 16 — Personīgā produktivitāte ar Microsoft Edge + Copilot Chat (praktiski)

## 🎯 Mērķis
Šī uzdevuma mērķis ir gūt **praktisku ieskatu**, kā Microsoft Edge iebūvētais **Copilot Chat** var palīdzēt ikdienā:
- ātri saprast (un “izvilkt”) būtisko no rakstiem,
- strādāt ar YouTube saturu (kopsavilkumi, precizējumi, laika atzīmes),
- izmantot vairākus pārlūka “tabus” kā kontekstu vienā sarunā,
- un kopumā — ietaupīt laiku informācijas apstrādē.

> Piezīme: iespējas var atšķirties atkarībā no Edge/Copilot versijas, reģiona un konta (bezmaksas/maksas, organizācijas iestatījumi).

---

## ✅ 0. Sagatavošanās
Tev vajadzēs:
- Microsoft Edge pārlūkprogrammu
- piekļuvi internetam
- (vēlams) austiņas, ja skatīsies YouTube

# 🧩 16.1. Izveido jaunu profilu Edge: **Personal**
## Kāpēc?
Papildus profils palīdz nodalīt:
- darbu / mācības / personīgo pārlūkošanu,
- vēsturi, sīkfailus, grāmatzīmes, pielāgojumus,
- un dažreiz arī Copilot kontekstu (atkarībā no iestatījumiem).

## ✅ Darbība
1. Atver **Microsoft Edge**.
2. Augšējā labajā stūrī atrodi profila ikonu (parasti aplītis ar burtu/attēlu).
3. Izvēlies: **Set up new profile** (vai līdzīgu opciju).
4. Izvēlies profila tipu: **Personal**.
5. Sekojot instrukcijām, pabeidz profila izveidi.

## ✅ Pārbaude
- [ ] Man ir jauns Edge profils ar nosaukumu / tipu **Personal**
- [ ] Es varu pārslēgties starp profiliem

---

# 🧪 16.2. Raksts pārlūkā + Copilot: kopsavilkums un precizējumi
## ✅ Mērķis
Atvērt jebkuru sev interesējošu mājaslapu (vēlams konkrētu rakstu) un ar Copilot palīdzību:
- iegūt kopsavilkumu,
- izvilkt specifisku informāciju,
- uzdot precizējošus jautājumus.

## ✅ Darbība
1. Atver kādu rakstu, piemēram:
   - ziņu portālu rakstu,
   - bloga ierakstu,
   - wiki lapu,
   - ceļojuma aprakstu,
   - instrukciju (how-to).
2. Atver **Copilot** Edge pārlūkā (parasti ikona augšējā labajā stūrī).
3. Uzdod 3–5 jautājumus par atvērtās lapas saturu (sk. piemērus zemāk).

### 💬 Jautājumu piemēri (kopē/ielīmē)
1)
```
Iedod šīs lapas īsu kopsavilkumu 5 punktos.
```

2)
```
Kādi ir 3 galvenie secinājumi no šī raksta un kam tie ir svarīgi?
```

3)
```
Izvelc no lapas konkrētus faktus/soļus un iedod tos kā checklist.
```

4)
```
Kas šajā rakstā ir neskaidrs vai pretrunīgs? Ko būtu vērts pārbaudīt?
```

5)
```
Ja man ir tikai 60 sekundes, ko man noteikti jāzina no šīs lapas?
```

## ✅ Pārbaude (mini-checklist)
- [ ] Copilot spēj sniegt kopsavilkumu par atvērtās lapas saturu
- [ ] Es varu uzdot precizējošus jautājumus un dabūt konkrētas atbildes
- [ ] Atbildes ir noderīgas, nevis “vispārīgas frāzes”

# ▶️ 16.3. YouTube video (angļu valodā): kopsavilkums, precizējumi, laika atzīmes, “Take a quiz”
## ✅ Mērķis
Iemācīties “apstrādāt” garāku YouTube saturu ātrāk:
- kopsavilkums,
- precizējoši jautājumi,
- konkrēts video laika brīdis (timestamp),
- un mini-eksāmens (quiz).

## ✅ Video izvēle
1. Atver **youtube.com**
2. Atrodi video **angļu valodā**, kas:
   - ir pietiekami garš (ieteicams 10–30+ min),
   - ir par tēmu, kas tev tiešām interesē (lai ir motivācija testēt).

## ✅ Darbība: kopsavilkums
Atver Copilot Chat un ieraksti:

```
Iedod šī video kopsavilkumu latviski 8–12 punktos.
```

## ✅ Darbība: precizējoši jautājumi
Izvēlies 2–4 un uzdod Copilot:

1)
```
Kādi ir 3 galvenie “takeaways” un kā tos var praktiski pielietot?
```

2)
```
Kādas ir 2 idejas, kuras video autors uzsver visvairāk? Pamato ar piemēriem no video.
```

3)
```
Kas video var būt strīdīgs vai ar ko vajadzētu būt uzmanīgam?
```

## ✅ Darbība: konkrēts laika brīdis (timestamp)
Tev jāizvēlas, **kāda informācija** tevi interesē (piem., “X metode”, “Y ieteikums”, “Z definīcija”), un tad pajautā:

```
Kad tieši video (laika brīdis) tiek izskaidrots [TAVA TĒMA]?
Iedod precīzu timestamp (piem., 12:34) un 1 teikuma kopsavilkumu, kas tur tiek pateikts.
```

> Ja Copilot nevar iedot precīzu laika brīdi, palūdz:
> - iedot aptuveno intervālu (piem., 10:00–14:00),
> - vai pasaki, lai tas uzdod 1 jautājumu precizēšanai (“kuru daļu tu domā?”).

---

## 🧠 16.3.1. Izveido “Take a quiz” (5 jautājumi)
Dažos Edge/Copilot skatījumos var būt poga **“+ Take a quiz”** (vai līdzīga). Mēs to izmantosim, lai pārbaudītu, vai tiešām saprati video.

### ✅ Darbība
1. Copilot čatā atrodi opciju: **+ Take a quiz**
2. Izvēlies to
3. Uzvednē (prompt) ieliec šādu tekstu (kopē/ielīmē):

```
Izveido quiz par šo YouTube video ar 5 jautājumiem.
Formāts:
- 5 multiple-choice jautājumi (A/B/C/D)
- pie katra jautājuma iedod pareizo atbildi un 1 teikuma skaidrojumu
- jautājumiem jāaptver dažādas video daļas (sākums/vidus/beigas)
Valoda: latviešu.
```

4. Izpildi quiz (patiess tests: atbildi pirms skaties “pareizo atbildi”).
5. Pieraksti rezultātu: `x/5`.

## ✅ Pārbaude (mini-checklist)
- [ ] Es dabūju kopsavilkumu par video
- [ ] Es uzdevu precizējošus jautājumus un dabūju sakarīgas atbildes
- [ ] Es mēģināju iegūt timestamp par interesējošo momentu
- [ ] Es izveidoju un izpildīju 5 jautājumu quiz

# 🧩 16.4. “Čatošana ar vairākiem tabiem” (Add tabs)
## ✅ Mērķis
Iemācīties pievienot sarunai kontekstu no vairākām atvērtām lapām (tabiem), lai Copilot var salīdzināt, atrast kopīgo vai atšķirīgo.

## ✅ Darbība
1. Atver vismaz **2–3 tabus**:
   - Vari izvēlēties saistītus (par vienu tēmu), piemēram:
     - 2 raksti par to pašu notikumu,
     - 1 raksts + 1 wiki + 1 bloga ieraksts.
   - Vai arī speciāli nesaderīgus (jautrībai):
     - recepte,
     - tehnisks raksts,
     - ceļojuma apraksts.
2. Atver **jaunu Copilot sarunu** (New chat).
3. Izvēlies: **+ Add tabs** (vai līdzīgu opciju).
4. Pievieno sarunai 2 vai vairāk no atvērtajiem tabiem.
5. Uzdod zemāk dotos “testa jautājumus”.

### 💬 3 jautri jautājumi testam (kopē/ielīmē)
1)
```
Kas šajos resursos ir kopīgs? Iedod 5 punktus (pat ja kopīgais ir netiešs).
```

2)
```
Radoši: kāpēc šos resursus nekad nevajadzētu lasīt vienlaicīgi? (Iedomājies, ka tas var sabojāt manu dienu.)
```

3)
```
Izveido “ideju kokteili”: paņem 1 ideju no katra resursa un apvieno to vienā praktiskā plānā 5 soļos.
```

> Ja pievienotie tabi ir tiešām nesaistīti, atbildes mēdz būt smieklīgākas — tas ir OK. Mērķis ir saprast, ka Copilot var “redzēt” vairākus kontekstus vienlaicīgi (ja funkcija pieejama).

## ✅ Pārbaude (mini-checklist)
- [ ] Es pievienoju sarunai vairākus tabus ar “Add tabs”
- [ ] Copilot spēj salīdzināt un sintezēt informāciju no tiem
- [ ] Es redzu praktisku pielietojumu (vai vismaz “aha, interesanti” efektu)
